## 技术架构现状

### 前端架构
- **框架**: React 19 + TypeScript
- **UI库**: Ant Design 5
- **路由**: React Router v6
- **状态管理**: Context API + Custom Hooks
- **HTTP客户端**: Axios

### 后端架构
- **语言**: Go 1.24.3
- **框架**: Gorilla Mux
- **数据库**: SQLite
- **认证**: JWT
- **文件存储**: 本地文件系统

### 项目结构
```
mini-web/
├── backend/              # Go后端
│   ├── cmd/server/      # 服务入口
│   ├── internal/        # 内部包
│   │   ├── api/         # API处理器
│   │   ├── model/       # 数据模型
│   │   ├── service/     # 业务逻辑
│   │   └── middleware/  # 中间件
│   └── data/           # 数据文件
├── frontend/           # React前端
│   ├── src/
│   │   ├── components/ # 可复用组件
│   │   ├── pages/      # 页面组件
│   │   ├── hooks/      # 自定义Hook
│   │   ├── services/   # API服务
│   │   └── contexts/   # React Context
│   └── public/         # 静态资源
└── docker-compose.yml  # 容器编排
```
